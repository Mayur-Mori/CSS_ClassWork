CSS FILTER

The filter property defines visual effects (like blur and saturation) to an element (often <img>).

  • filter: blur(5px);
  • filter: brightness(0.4);
  • filter: contrast(200%);
  • filter: drop-shadow(16px 16px 20px blue);
  • filter: grayscale(50%);
  • filter: hue-rotate(90deg);
  • filter: invert(75%);
  • filter: opacity(25%);
  • filter: saturate(30%);
  • filter: sepia(60%);

Note: To check the brightness property please hover on 2nd image.
